Conecte ao Supabase para sincronizar entre dispositivos
Conexão Supabase
Sem Supabase, os dados ficam salvos só neste navegador. Conectando, você acessa de qualquer dispositivo e os dados ficam na nuvem.
Seus dados atuais do localStorage não serão perdidos — eles serão enviados ao Supabase na primeira conexão.
Sincronização
Como criar o banco de dados
  1. Acesse supabase.com → criar conta gratuita
  2. Clique em New project → dê um nome → aguarde (~2 min)
  3. No menu: SQL Editor → New query
  4. Cole o conteúdo do arquivo supabase_schema.sql → clique em Run
  5. Vá em Settings → API e copie a Project URL e a chave anon public
  6. Cole os valores nos campos acima e clique em Salvar e conectar
Nota: As chaves ficam salvas no seu navegador (localStorage), não no código do GitHub. Cada dispositivo precisa configurar uma vez.
