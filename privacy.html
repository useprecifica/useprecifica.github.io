Política de Privacidade

Última atualização: junho de 2026

Esta Política de Privacidade descreve como o Precifica ("nós", "nosso") coleta, usa e protege as informações dos usuários do aplicativo web e da extensão para Chrome disponíveis em useprecifica.github.io.

1. Informações que coletamos

Coletamos apenas as informações necessárias para o funcionamento do serviço:

  • Dados de conta: endereço de e-mail e senha (armazenados de forma segura via Supabase Auth, a senha nunca é armazenada em texto puro).
  • Dados de anúncios: títulos, códigos, preços e custos dos seus anúncios no Mercado Livre e Shopee, inseridos manualmente ou importados via extensão Chrome.
  • Token de acesso do Mercado Livre: quando você conecta sua conta ML à extensão, armazenamos o token de acesso localmente no seu navegador (via Chrome Storage) para importar seus anúncios. Esse token nunca é enviado a servidores de terceiros além dos próprios servidores do Mercado Livre e do Supabase.
  • Dados de uso: nenhum dado de navegação, rastreamento de comportamento ou analytics de terceiros é coletado.

2. Como usamos suas informações

  • Para calcular e exibir a margem dos seus anúncios
  • Para salvar seus anúncios, custos e simulações na sua conta
  • Para processar pagamentos e gerenciar seu plano (Free ou Pro) via Mercado Pago
  • Para enviar e-mails transacionais (confirmação de cadastro, redefinição de senha)

3. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros para fins de marketing. Seus dados podem ser compartilhados apenas com:

  • Supabase: plataforma de banco de dados e autenticação que hospeda seus dados com criptografia em repouso.
  • Mercado Pago: para processar pagamentos de assinatura Pro. Só transmitimos seu e-mail para facilitar a identificação da assinatura.

4. Extensão Chrome — permissões

A extensão Precifica solicita as seguintes permissões e as utiliza exclusivamente para:

  • storage: salvar sua sessão de login e token ML localmente no navegador
  • tabs: abrir uma aba temporária do Mercado Livre para realizar buscas de produtos (a aba é fechada automaticamente)
  • activeTab: ler dados da página do Alibaba/1688 quando você clica no botão Comparar
  • host_permissions (mercadolivre.com.br, alibaba.com, 1688.com): injetar o painel flutuante nessas páginas

A extensão não lê, coleta ou transmite o conteúdo de nenhuma outra página que você visitar.

5. Retenção de dados

Seus dados ficam armazenados enquanto sua conta estiver ativa. Você pode solicitar a exclusão completa da sua conta e dados a qualquer momento pelo e-mail abaixo.

6. Segurança

Utilizamos RLS (Row Level Security) do PostgreSQL para garantir que cada usuário acesse apenas seus próprios dados. Toda comunicação é feita via HTTPS com certificados SSL.

7. Seus direitos

De acordo com a LGPD (Lei Geral de Proteção de Dados — Lei nº 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou incorretos
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento a qualquer momento

8. Contato

Para exercer seus direitos ou tirar dúvidas sobre esta política, entre em contato:
contato@useprecifica.com.br

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por e-mail.
